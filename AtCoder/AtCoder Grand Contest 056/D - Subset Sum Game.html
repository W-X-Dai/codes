D - Subset Sum Game Editorial /

Time Limit: 2 sec / Memory Limit: 1024 MB

配点 : 160016001600 点

問題文

黒板に NNN 個の整数が書かれており，そのうち iii 番目の整数は AiA_iAi​ です． なお，NNN は偶数です． また，整数 L,RL,RL,R が与えられます．

Alice と Bob がゲームをします． 二人は Alice からはじめて交互に手番をプレイします． 各手番では，プレイヤーは黒板に書かれている数を一つ選んで消します．

NNN ターン後にゲームが終了します． ここで，Alice が消した整数の総和を sss とします． L≤s≤RL \leq s \leq RL≤s≤R であれば Alice の勝利，そうでなければ Bob の勝利です． 両者が最適に行動した時，どちらが勝つか求めてください．

制約

  • 2≤N≤50002 \leq N \leq 50002≤N≤5000
  • NNN は偶数
  • 1≤Ai≤1091 \leq A_i \leq 10^91≤Ai​≤109
  • 0≤L≤R≤∑1≤i≤NAi0 \leq L \leq R \leq \sum_{1 \leq i \leq N} A_i0≤L≤R≤∑1≤i≤N​Ai​
  • 入力される値はすべて整数である

入力

入力は以下の形式で標準入力から与えられる．

NNN LLL RRR
A1A_1A1​ A2A_2A2​ ⋯\cdots⋯ ANA_NAN​

出力

Alice が勝つ場合は Alice，Bob が勝つ場合は Bob と出力せよ．

Problem Statement

There are NNN integers written on a blackboard. The iii-th integer is AiA_iAi​. Here, NNN is even. Additionally, integers LLL and RRR are given.

Alice and Bob will play a game. They alternately take turns, with Alice going first. In each turn, the player chooses a number written on the blackboard and erases it.

The game will end in NNN turns. Then, let sss be the sum of the integers erased by Alice. If L≤s≤RL \leq s \leq RL≤s≤R, Alice wins; otherwise, Bob wins. Find which player will win when both players play optimally.

Constraints

  • 2≤N≤50002 \leq N \leq 50002≤N≤5000
  • NNN is even.
  • 1≤Ai≤1091 \leq A_i \leq 10^91≤Ai​≤109
  • 0≤L≤R≤∑1≤i≤NAi0 \leq L \leq R \leq \sum_{1 \leq i \leq N} A_i0≤L≤R≤∑1≤i≤N​Ai​
  • All values in input are integers.

Input

Input is given from Standard Input in the following format:

NNN LLL RRR
A1A_1A1​ A2A_2A2​ ⋯\cdots⋯ ANA_NAN​

Output

If Alice wins, print Alice; if Bob wins, print Bob.

Sample Input 1 Copy

Copy
4 5 6
3 1 4 5

Sample Output 1 Copy

Copy
Alice

In this game, Alice will always win. Below is a possible progression of the game.

  • Alice erases 111.
  • Bob erases 444.
  • Alice erases 555.
  • Bob erases 333.

Here, the sum of the integers erased by Alice is 666. Since L≤6≤RL \leq 6 \leq RL≤6≤R, Alice wins.
