問題文

0, 1 からなる長さ NNN の文字列 sss を作ることを考えます． ここで，sss は MMM 個の条件を満たす必要があります． iii 番目の条件は整数 Li,RiL_i,R_iLi​,Ri​ (1≤Li<Ri≤N1 \leq L_i < R_i \leq N1≤Li​<Ri​≤N) で表されます． これは，sss の LiL_iLi​ 文字目から RiR_iRi​ 文字目までを見たときに，そこに含まれる 0 の個数と 1 の個数が等しい必要があることを意味します．

すべての条件を満たす中で辞書順最小の sss を求めてください． なお，問題の制約より，条件を満たす sss が必ず存在することが証明できます．

制約

  • 2≤N≤1062 \leq N \leq 10^62≤N≤106
  • 1≤M≤2000001 \leq M \leq 2000001≤M≤200000
  • 1≤Li<Ri≤N1 \leq L_i < R_i \leq N1≤Li​<Ri​≤N
  • (Ri−Li+1)≡0mod  2(R_i-L_i+1) \equiv 0 \mod 2(Ri​−Li​+1)≡0mod2
  • (Li,Ri)≠(Lj,Rj)(L_i,R_i) \neq (L_j,R_j)(Li​,Ri​)=(Lj​,Rj​) (i≠ji \neq ji=j)
  • 入力される値はすべて整数である

入力

入力は以下の形式で標準入力から与えられる．

NNN MMM

L2L_2L2​ R2R_2R2​

Problem Statement

Consider making a string sss of length NNN consisting of 0 and 1, where sss must satisfy MMM conditions. The iii-th condition is represented by integers LiL_iLi​ and RiR_iRi​ (1≤Li<Ri≤N1 \leq L_i < R_i \leq N1≤Li​<Ri​≤N). This means that there should be equal numbers of 0 and 1 between the LiL_iLi​-th and RiR_iRi​-th characters (inclusive) of sss.

Find the lexicographically smallest string sss that satisfies all conditions. It can be proved that the Constraints of the problem guarantee the existence of sss that satisfies the conditions.

Constraints

  • 2≤N≤1062 \leq N \leq 10^62≤N≤106
  • 1≤M≤2000001 \leq M \leq 2000001≤M≤200000
  • 1≤Li<Ri≤N1 \leq L_i < R_i \leq N1≤Li​<Ri​≤N
  • (Ri−Li+1)≡0mod  2(R_i-L_i+1) \equiv 0 \mod 2(Ri​−Li​+1)≡0mod2
  • (Li,Ri)≠(Lj,Rj)(L_i,R_i) \neq (L_j,R_j)(Li​,Ri​)=(Lj​,Rj​) (i≠ji \neq ji=j)
  • All values in input are integers.

Input

Input is given from Standard Input in the following format:

NNN MMM
L1L_1L1​ R1R_1R1​
L2L_2L2​ R2R_2R2​
⋮\vdots⋮
LML_MLM​ RMR_MRM​
